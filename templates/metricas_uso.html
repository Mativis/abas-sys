Como funciona:

O sistema calcula automaticamente que a próxima troca de óleo deve ser feita:

  • Veículos: A cada 10.000 KM rodados após a última troca
  • Máquinas/Equipamentos: A cada 350 horas de uso

Os indicadores mostram o status baseado no critério mais crítico.

Atenção:

Para equipamentos, use a mesma identificação nos checklists para que o sistema possa acompanhar o horímetro atual.
